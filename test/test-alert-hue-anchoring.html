What is Hue Anchoring?

Hue anchoring is a color transformation technique that pulls all colors toward a specific hue range, creating a cohesive "color personality" for each alert mode. This matches the polished aesthetic of modern Star Trek interfaces (Discovery, Picard, Lower Decks) where alert modes have a distinct, unified color scheme.

How it works:
1. Colors are first rotated toward the alert mode's target hue
2. Hue anchoring then pulls colors outside the target range back toward the range boundaries
3. Colors remain distinguishable while feeling cohesively themed

Benefits of Hue Anchoring

  • Cohesive Visual Identity: Each alert mode has a distinct "color personality"
  • Modern LCARS Aesthetic: Matches polished look of Discovery/Picard interfaces
  • Preserved Distinctiveness: Colors remain distinguishable within range
  • Fine-Tunable: Adjust centerHue, range, and strength per alert mode
  • Backward Compatible: Green/gray/black alerts unchanged

Technical Implementation

The hue anchoring algorithm calculates the shortest distance on the circular color wheel (handling 360°→0° wrapping correctly), then interpolates colors outside the target range back toward the range edge using a configurable strength parameter.

Function: applyHueAnchor(hue, anchor)
