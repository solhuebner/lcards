## How to Contribute

Contributions to LCARdS are welcome. Please open an issue or pull request on [GitHub](https://github.com/snootched/lcards).

## Conventions

- Follow existing code style (see `.editorconfig` and ESLint config)
- Include JSDoc for all public methods and classes
- Document breaking changes in PR descriptions
- Reference `src/cards/*` for current card implementation patterns
