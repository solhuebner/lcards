# Themes & Colours

LCARdS uses a token-based theming system managed by the **Theme Manager** (`window.lcards.core.themeManager`).

## Overview

Theme tokens define colours, spacing, border radii, and other visual properties that are applied consistently across all cards. Tokens can be referenced in any card field using the `{theme:path.to.token}` template syntax.
