MSD Configuration Studio - Phase 2 Test

Test Overview

This page tests the Phase 2 implementation of the MSD Configuration Studio:

  • Base SVG Tab: SVG source, viewBox, and filters configuration
  • Anchors Tab: Create, edit, delete anchors with visual placement
  • Place Anchor Mode: Click canvas to place anchors
  • Coordinate Helpers: Grid overlay and snap-to-grid
  • Live Preview: Real-time preview with debug visualizations

Manual Testing Checklist

Base SVG Tab

  • ☐ SVG source input updates config.msd.base_svg.source
  • ☐ Builtin templates work (e.g., builtin:ncc-1701-a)
  • ☐ ViewBox mode toggle switches between auto and custom
  • ☐ Custom viewBox inputs work correctly
  • ☐ Filter preset selector updates config.msd.base_svg.filter_preset
  • ☐ Custom filters toggle enables filter inputs
  • ☐ Custom filter sliders update values
  • ☐ Preview updates when base_svg changes

Anchors Tab

  • ☐ "Add Anchor" button opens anchor form dialog
  • ☐ Anchor form validates name (required, no duplicates)
  • ☐ Position inputs (X, Y) work correctly
  • ☐ Unit selector works (px, %, vb)
  • ☐ "Save" button creates new anchor in config.msd.anchors
  • ☐ Edit button opens form with existing anchor values
  • ☐ Delete button removes anchor from config
  • ☐ Anchor list displays all anchors

Place Anchor Mode

  • ☐ "Place on Canvas" button enters Place Anchor Mode
  • ☐ Mode toolbar shows "Place Anchor" as active
  • ☐ Clicking preview canvas opens anchor form
  • ☐ Click coordinates converted to viewBox coordinates
  • ☐ Position pre-filled in anchor form from click
  • ☐ Auto-generated anchor names (anchor_1, anchor_2, etc.)
  • ☐ Mode exits after opening anchor form

Coordinate Helpers

  • ☐ "Show Grid Overlay" checkbox toggles grid
  • ☐ Grid spacing selector works (10px, 20px, 50px, 100px)
  • ☐ Grid renders in preview when enabled
  • ☐ "Snap to Grid" checkbox toggles snapping
  • ☐ Snap applies to anchor placement clicks

Live Preview

  • ☐ Preview shows "No base SVG configured" when empty
  • ☐ Preview renders MSD card when base_svg present
  • ☐ Preview updates with 300ms debounce on config changes
  • ☐ Preview shows anchor markers from config.msd.anchors
  • ☐ Preview shows grid overlay when enabled
  • ☐ Manual refresh button works
