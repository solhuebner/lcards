## Generated File Structure

### Editor Class (example: chart)

```javascript
/**
 * Chart Card Editor
 * 
 * Visual editor for LCARdS Chart Card.
 * Generated by tools/create-editor.js on 2025-12-30
 * 
 * @extends LCARdSBaseEditor
 */

import { html } from 'lit';
import { LCARdSBaseEditor } from '../base/LCARdSBaseEditor.js';
import { LCARdSFormFieldHelper as FormField } from '../components/shared/lcards-form-field.js';
import '../components/shared/lcards-form-section.js';

export class LCARdSChartEditor extends LCARdSBaseEditor {

    constructor() {
        super();
        this.cardType = 'chart';
    }

    _getTabDefinitions() {
        return [
            { label: 'General', content: () => this._renderGeneralTab() },
            { label: 'Style', content: () => this._renderStyleTab() },
            { label: 'Advanced', content: () => this._renderAdvancedTab() },
            ...this._getUtilityTabs()
        ];
    }

    _renderGeneralTab() {
        return html`
            <lcards-form-section 
                header="Basic Configuration"
                description="Core card settings">
                
                ${FormField.renderField(this, 'entity', {
                    label: 'Entity',
                    helper: 'Select the primary entity for this card'
                })}
                
                ${FormField.renderField(this, 'name', {
                    label: 'Name',
                    helper: 'Display name for the card (optional)'
                })}
                
                <!-- TODO: Add chart-specific fields here -->
                
            </lcards-form-section>
        `;
    }

    // ... more tab renderers
}

customElements.define('lcards-chart-editor', LCARdSChartEditor);
```

## Key Features Demonstrated

### ✅ Validation

- Checks for duplicate editors
- Validates card type format (lowercase, alphanumeric + hyphens)
- Ensures at least one tab is selected

### ✅ Smart Defaults

- Auto-generates label from card type
- Selects common tabs by default
- Recommends "Standard" template
- Defaults to generating schema

### ✅ Template System

- **Minimal**: 100 lines total (single tab)
- **Standard**: 200 lines total (3 tabs)
- **Advanced**: 300+ lines (4 tabs + advanced features)
- **Clone**: Full copy of existing editor

### ✅ Production-Ready Output

- Proper JSDoc comments
- TODO markers for customization
- FormField integration
- Schema with x-ui-hints
- Choose selectors for complex fields
- Self-registering custom element

## Next Steps After Generation

1. **Customize Schema**: Add card-specific properties
   ```bash
   open tools/schema-editor.html
   ```

2. **Add Tab Content**: Enhance tab renderers with specific fields
   ```javascript
   ${FormField.renderField(this, 'chart_type', {
       label: 'Chart Type',
       helper: 'Line, bar, area, or pie'
   })}
   ```

3. **Create Card Class**: Implement the actual card
   ```javascript
   // src/cards/lcards-chart.js
   import '../editor/cards/lcards-chart-editor.js';
   ```

4. **Build & Test**:
   ```bash
   npm run build
   # Copy to HA, reload browser
   ```

## Tips

- Use **Standard** template for most cards
- Use **Clone** when creating similar variants
- Use **Minimal** for quick prototypes
- Always generate schema (easier to edit than create from scratch)
- Use Schema Editor tool for complex properties
- Add TODO comments as you customize
